# react_datagrid_app
Solidity Smart Contract, Web3 Application

### Features
Smart Contract Compile, Test, Deploy (Ropsten Testnet)
Web3.js + React + Kendo UI

### Install

    npm install

### Smart Contract Compile 

    npm run compile

### Smart Contract Test

    npm run test

### Smart Contract Deploy

    npm run deploy

### React Application Start

    npm start

### Technologies

- Solidity
- Chai
- Hardhat
- Ropsten Testnet
- Typescript
- React
- Web3.js
- Kendo React
